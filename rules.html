 შეკვეთა

სამომხმარებლო წესები:

    ზოგადი წესები:

  1. ჩვენი სერვისით ( ჰოსტინგით ) სარგებლობა ნიშნავს, რომ თქვენ ავტომატურად ეთანხმებით წესებს.
  2. აღნიშნული წესები წარმოადგენს Linuxserv -ს ოფიციალურ სამომხარებლო წესებს და მათი დაცვა სავალდებულოა ყველა მომხმარებლისთვის.
  3. წესების დაღვევის შემთხვევაში ადმინისტრაცია მოახდენს შესაბამის რეაგირებას და თავად გადაწყვიტავს ნებისმიერ საკითხს.

    4. სერვერის - შეკვეთა / მფლობელი / გაუქმება:

  4. 1. მომხმარებლის მიერ სერვერის შეკვეთა ხდება მხოლოდ ვებ-გვერდის მეშვეობით.
  5. სერვერის მფლობელად ავტომატურად ითვლება პიროვნება ვინც პირველადი შეკვეთა გააკეთა სერვერზე.
  6. მფლობელს შეუძლია თავისი სურვილისამებრ სერვერის: გაყიდვა - გაუქმება.
  7. ასევე შესაძლებელია სერვერის მფლობელის შეცვლა ( გადაფორმება ).
  8. სერვერის პარამეტრები, ადმინ პანელი და FTP -ანგარიში გაიცემა მხოლოდ სერვერის მფლობელზე.
  9. სერვერის პარამეტრების რედაქტირება ხდება მხოლოდ სერვერის მფლობელის მოთხოვნით.
  10. სერვერის გაყიდვის დროს აუცილებელია მიმდინარე მფლობელმა მოახდინოს Linuxserv - ადმინისტრაციის ინფორმირება და გადაფორმდეს სერვერი ახალ მფლობელზე, წინააღმდეგ შემთხვევაში Linuxserv - ადმინისტრაცია იხსნის ყოველგვარ პასუხისმგებლობას.

    11. სერვერის მოხმარების პირობები:

  11. სერვერზე სასტიკად აკრძალულია ბოტების გამოყენება. ( გარდა სპეციალური პლუგინისა, რომელიც მოიცავს 3 სპექტატორს )
  12. დაუშვებელია სერვერის წაშლა GameTracker.com - ს რეიტინგიდან.
  13. სერვერზე აკძალულია პორნოგრაფიის, ეროტიკის და სხვა საქართველოს კანონმდებლობით აკრძალული მასალების განთავსება.
  14. დაუშვებელია სერვერზე გადამისამართების დაყენება და სხვა სერვერზე პორტირება მომხმარებლების.
  15. Luxserv - ადმინისტრაცია არ ერევა სერვერების მფლობელების და მათ შიდა საქმიანობაში, აქედან გამომდინარე პრეტენზიები კონკრეტული ადმინის ან ადმინისტრატორების მიმართ არ განიხილება. ( მათ შორის, არც ადმინის გაყიდვასთან დაკავშირებული საკითხები ).
  16. სერვერის მფლობელს შეუძლია მოითხოვოს რამდენჯერმე ( მაქსიმუმ 2-3 ჯერ ) თავისი სერვერის პარამეტრების შეცვლა, მაგალითად: ადმინ პანელის ან FTP - ანგარიშის.
  17. სერვერის მფლობელს ასევე შეუძლია სერვერის გაყიდვის დროს ( კონკრეტული მისამართის ) გადაიხადოს მხოლოდ სერვისის - ჰოსტინგის საფასური, ხოლო დანარჩენი თანხა სარგებლის სახით დაიტოვოს.
  18. სერვერის თანხის გადაუხდელობის შემთხვევაში Luxserv -ს ადმინისტრაცია იტოვებს უფლებას, ჩამოართვას სერვერი მიმდინარე მფლობელს და მოახდინოს სერვერის მისამართის სხვაზე რეალიზაცია. ( სერვისის - ჰოსტინგის დავალიანების დაფარვის მაქსიმალური ვადაა 3 დღე ).
  19. Linuxserv -ს ადმინისტრაციას აქვს უფლება ნებისმიერ დროს მოითხოვოს სერვერის მფლობელისგან, ნებისმიერი პლაგინის თუ მოდულის ან სხვა ტექნიკური საკითხის - შეცვლა, გაუქმება და სხვა...
  20. Linuxserv -ს ადმინისტრაციას აქვს უფლება, სამომხმარებლო წესების დამრღვევი პირის თუ სერვერის მფლობელის მიმართ, გაატაროს ნებისმიერი სახის მოქმედება და გადაწყვეტილება. ( მათ შორის - სერვერის ჩამორთმევა, სერვერის თანხის დაკავება, სრულ ჰოსტინგზე ბანის დადება და სხვა.. )
